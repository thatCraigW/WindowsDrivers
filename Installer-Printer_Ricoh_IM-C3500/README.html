Getting Started

Installing the printer driver

The installation procedure requires an administrator account or the ability to "Run as Administrator" in the Windows operating system.

Prior to installation, be sure to close all applications before installing the driver.
For installing the driver by using the "Add a Printer" feature in Windows, please refer to that operating system's Help section. 

    1.   Run "RV_SETUP.exe", which is located in the same folder as this readme.
    2.   The "End User License Agreement" dialog will appear. If you agree with the terms and conditions, select "Agree" and click "Next".
    3.   The "Select a setup method" dialog will appear. 
    4.   To display network-connected devices, select "Network" and click "Next", proceed to Step 5a. 
         If a USB connection is desired, select "USB" and click "Next". Proceed to Step 5b. ("USB" might not be selectable depending on the device type.) 
    
    - Network -
    5a.  The "Select a device" dialog will appear. Select the device from the list of network-connected devices and click "Next". Then, proceed to Step 10a. 
         To change the driver name and/or the device name, select the "Manually enter the settings" checkbox. In that case, proceed to Step 9a. 
    6a.  If the device cannot be found, click "Click if you cannot find the device". The "Select a search method" dialog will appear.
             a. To specify the device's IP address, select "Specify device IP address" and click "Next". Then, proceed to Step 7a.
             b. If the device's port has already been configured or if a new port needs to be configured, select "Specify device name" and click "Next". Then, proceed to Step 8a.
    7a.  The "Search by IP address" dialog will appear. Enter the IP address and click "Search" to search for the device. If the device is found, click "Next". Then, proceed to Step 10a.
    8a.  The "Select a device" dialog will appear. Select a device and click "Next".
    9a.  The "Printer Preference" dialog will appear. Choose one of the following methods to set up the port: 
             a. If the port has already been registered, select the port to "Port Name".
             b. To specify a new port, click "Add Port". The "Add Port" dialog will appear and a new port can be added. After that new port has been added, the name of it will be selected.
    10a. When installation has completed, the "Complete" dialog will appear. Click "Finish".
    
    - USB -
    5b.  The "Select Device" dialog will appear. Select the device name or connect the device with USB cable, and click "Next". 
    6b.  The "Install" dialog will appear. When installation has completed, the dialog changes automatically.
    7b.  When the "Connection with device" dialog appears. 
             a. If the device is not running and not connected with USB cable, turn on the device and connect with USB cable.
             b. If the device is running and connected with USB cable, wait a moment.
    8b. When the device is detected, the dialog changes automatically.
    9b. The "Complete" dialog will appear. Click "Finish".

Upgrading the printer driver

If the printer driver is already installed, upgrade the printer driver.
Prior to installation, be sure to close all applications before upgrading the driver.
Be sure to close all applications before installing or upgrading the driver.

  1. Run "RV_SETUP.exe", which is located in the same folder as this readme.
  2. When the "End User License Agreement" dialog appears, if you agree with the terms and conditions, select "Agree" and click "Next"
  3. When the "Select a setup method" dialog appears, click "Update Driver"
  4. When the "Select the driver" dialog appears and upgradable targets are displayed, confirm and select "Update" and click "Next".
  5. When the "Complete" dialog is appears, select "Yes, restart now" and click "Finish" and restart your computer.

Other important points

[General Issue]

  • After the installations, if the printer setting dialogs seem abnormal, delete the printer driver and install it again.
  • To be available device's functions, set up Accessories. Accessories settings are required if the bi-directional communication is not activated. It might not necessary depending on the printer driver type.
  • Installing with this installer might replace RICOH printer driver's print processor settings if they are already installed to system and customized. In such case, customize settings again or install with [Add a Printer] function provided by Windows.
  • In the [Printer Preference] dialog, printer port setting might be empty. Select port and click "Next" to continue installation.

[OS/Windows Related Issues]

  • Log on as Administrators to use this installer program.
  • When the computer is turned on or logged on, following messages might be displayed. Their descriptions are depends on OS type. If these messages are displayed, click [Cancel].
      Å@
    • a.[Found New Hardware] Å@
    • b.[Found New Hardware Wizard] Å@
    • c.[Add Hardware Wizard]
  • During printer driver's installation, confirmation dialog of digital signature might be displayed. But there is no problem for its function, so continue installation.
  • If a Microsoft minidriver installer is on the computer, when a plug-and-play printer is connected to the computer, the OS might install its minidriver automatically.
  • Do not use this installer on the environment of Microsoft Cluster Server. During the installation procedure, spooler service will be stopped temporary and this might touch other services. Install with [Add a Printer] function that provided by Windows surely. Depending on the driver's version, this installer might be unavailable and following message will be displayed.
